# 运算符
## 算数运算符
number 运算：
1. +-*/
2. 余数：%
3. 指数：**
4. 自增自减: a++ b--  ++a  --b  (少用,只在for循环中使用)
   数字在前 得到的值为运算前
   用 a += 1
5. 求值运算符 +x  -x
string 运算：
只支持：'123' + '456'
## 比较运算符
1. ==：两个等于尽量不要用，而用三个等于
2. ===:三个等于
3. !=:感叹号+等号  
4. !==:感叹号+两个等号     Nan !== Nan  []!== [] {}!=={}
